# MedSort
A small python TKinter program to manage medical indents

## Components
MedSort.py -> Main window GUI
Searcher.py -> Algo for searcher

SearchSetup.py -> Refresh DB GUI
SearchFileMaker.py -> Algo for DB Maker


## Helpful Commands
